#UseAme
É super fácil e seguro usar a Ame para fazer transações.

Como criar sua conta

Baixe o app Ame Digital.

Com poucos dados você já cria a sua conta.

E digitalizando, você cadastra seu cartão.

Prontinho, você já tem uma conta Ame. <3

Como pagar suas compras

Escolha Ame como forma de pagamento nas lojas parceiras.

Aponte seu celular para o QR Code que vai aparecer na tela.

É pa-pum. Já está pago. ;)

Como fazer transferências

Para receber: gere um Qr Code com o valor que tem para receber.

Quem for te pagar só vai precisar apontar o celular para o seu. Feito!

Para transferir: digite o valor que deve ser transferido.

Escolha se será por saldo ou cartão.

E selecione por contato para quem você vai enviar.

Resolvido, dívida paga!
